# 🏍️ Elite Motor Work's

A modern React-based superbike showcase website featuring premium motorcycles from top brands like BMW, Ducati, Kawasaki, Honda, Aprilia, and Suzuki.

---

## 📌 Features

* Dynamic greeting based on time of day
* Beautiful hero section with superbike background
* Responsive bike showcase grid
* Like/Unlike functionality using React Hooks
* View Details button linking to official bike pages
* Clean card-based UI design
* Fully built using React Components

---

## 🚀 Technologies Used

* React JS
* JavaScript (ES6)
* CSS3
* HTML5

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/elite-motor-works.git
```

2. Navigate to project folder

```bash
cd elite-motor-works
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```

## 🎯 Functionality

### Dynamic Greeting

The application displays a custom greeting depending on the current time:

* ☀️ Good Morning Rider!
* 🔥 Good Afternoon Speed Lover!
* 🌙 Good Evening Bike Enthusiast!

### Like Button

Users can:

* Click 🤍 to like a bike
* Toggle to ❤️ when selected

### View Details

Each bike card contains a button that redirects users to the official bike information page.
